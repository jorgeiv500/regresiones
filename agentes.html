🤖 Agentes Inteligentes

¿Qué es un agente?

Un agente percibe su entorno mediante sensores y actúa sobre él usando actuadores.

Robot Aspiradora

Sensores de polvo, motores

Termostato

Sensor de temperatura → calefacción

Asistente Virtual

Entrada de voz → acciones

Agente de Juego

Perceptos visuales → movimiento

🧠 Función del agente

Es una función matemática que toma los perceptos y produce una acción: f(perceptos*) → acción.

💻 Programa del agente

Es el código que implementa la función del agente. Ejemplo:

def programa_del_agente(percepto):
    if percepto == "temperatura_baja":
        return "encender_calefacción"

🏗️ Tipos de agentes

Reactivo Simple

Actúa según el percepto actual

Con Estado

Recuerda información pasada

Basado en Objetivos

Planifica para alcanzar una meta

Basado en Utilidad

Elige la acción más valiosa
